# will you go out with me? 🥺

A cute single-page web app inspired by Ally Codes' viral "i have a very important question" prompt.

Four screens:

1. **The ask** — "hey… i have a very important question for you" with a Yes button and a No button that runs away when you try to click it.
2. **Dream date** — pick from 6 date types (fancy dinner, movie night, picnic, mini golf, café, artsy adventure).
3. **When are you free** — pick from 4 timing options.
4. **Confirmation** — celebration screen with a summary of the choices and floating hearts.

## Run locally

Just open `index.html` in a browser — no build step.

```bash
# or serve it
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Tech

- Vanilla HTML/CSS/JS — no frameworks, no dependencies.
- Google Fonts: Caveat (handwritten title) + Inter (body).
- Hosted on GitHub Pages.

## Files

- `index.html` — markup for all 4 screens
- `styles.css` — pink/cream gradient theme, card animations, runaway-button styling
- `script.js` — screen navigation, runaway-no logic, option selection, confetti hearts

Enjoy 💕
